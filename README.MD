### 内容
D&D5版システム用の翻訳ファイルよび辞典が追加されます。追加機能として既存の英語の辞典を消して日本語版のみを見えるようにしてくれます（SRDのみ、日本語の無いものは英語のままで残る）

このモッドを使用するには先にモッドとしてインストールし、ゲーム内でモッドを有効化する必要があります。
